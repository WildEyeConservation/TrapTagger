Camera Trap Data Unit Counts

The data unit counts summary provides a summary of the number of data units captured per camera trap. The data unit can be specified as either clusters, images, sightings, or x time to independence. The data unit counts summary is useful for determining the total number of unit counts per camera trap. A date range can be specified to limit the data used in the analysis. A camera trap can be defined as either a site, camera, or site group.

Please click the Generate Results button to generate the summary results. The chart can be downloaded as a PNG by clicking the Export button. The chart display can be modified in Chart Options to how you would like it do be displayed when exported. Please note the summary results may take a while to generate if you have a large amount of data.
