Capture-Recapture Results

Capture-recapture analysis provides abundance estimates for a species. The results table displays these estimates for multiple models. It's important to note that capture-recapture is performed at the individual level and doesn't rely on spatial information for calculation.

Key aspects of the capture-recapture analysis include:

  • Observation Window: The observation window, specified in days, represents the duration of a sampling occasion.
  • Species: You can apply the analysis to the selected species or all species in the dataset.

The capture-recapture analysis is performed using an R script. You can access the script by clicking the View R Script button. Additionally, you have the option to download the data used in the analysis as a CSV file by clicking Download CSV. The R script includes a function that allows you to load the CSV file into R for further analysis or customization.

The capture-recapture analysis relies on the following R package:

  • Rcapture: This package is used to create capture-recapture models and calculate abundance estimates for species.

Please be aware that individual identification is crucial for accurate capture-recapture analysis. To initiate the analysis, click the Generate Results button. Keep in mind that generating capture-recapture results may take some time, so refrain from navigating away from the page or refreshing it during the process.
