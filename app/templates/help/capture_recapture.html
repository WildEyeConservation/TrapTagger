Capture-Recapture Results

Capture-Recapture provides abundance estimates for a species. The capture-recapture table showcase the abundance estimates for multiple models. Capture-recapture is done on an individual level. Capture-recapture do not use any spacial information to calculate the estimates.

The capture-recapture analysis can be filtered by a date range to only include data from a specific time period. The observation window is specified in days and is the number of days in a sampling occasion. Capture-recapture can be applied to one or multiple species as well as one or multiple sites. The analysis can be applied to one or multiple species as well as one or multiple sites.

The capture-recapture analysis was calculated using an R script. The script can be viewed by clicking the View R Script button. The data used in the analysis can be downloaded as a CSV file by clicking the Download CSV button. A function is created in the R script where the CSV file can be loaded into R and the analysis can be done using the function if further analysis is required or modifications to the analysis is needed for your own purposes.

The capture-recapture analysis is based on the following packages:

  • Rcapture

    • The Rcapture package is used to create capture-recapture models and to calculate the abundance estimates of a species.

Please note that capture-recapture is done on an individual level so individual id needs to be completed for a more accurate analysis. To run the analysis click the Generate Results button. Please note the capture-recapture results may take a while to generate. Do not navigate away from the page or refresh the page while the results are being generated.
