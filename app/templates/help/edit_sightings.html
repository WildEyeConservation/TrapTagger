Welcome to the sightings editor. Here you are able to edit the detected animal sightings in your images. Use add, edit, and delete modes to add, edit and delete sightings on an image before commiting those changes to the database. These functionalities can either be accessed using the toolbar on the left-hand side of the image, or by using the hotkeys provided below. You can also change the label associated with a particular sighting by right-clicking on the sighting, and selecting your desired label from the list provided. Should an unexpected species occur in an image, you can access all your labels, in hierarchical order, by selecting the plus option. As previously, you have access to a number of image-identification aids such as image brightness, contrast, and saturation. You can also zoom into the image.

To update the labels of multiple sightings at once, hold the Ctrl key while clicking on each sighting, then choose the appropriate label on the panel. If no sightings are selected and you apply a label, that label will automatically be assigned to all sightings in the image. This makes it easier to quickly label the majority of sightings and then correct the minority with their appropriate labels.

Hotkeys:

  • A: Add sighting. Press again to cancel and exit mode.
  • E: Edit sightings. Press again to cancel and exit mode.
  • D: Delete sightings. Press again to cancel and exit mode.
  • Esc: Cancel and exit current mode.
  • F: Finish making changes in current mode.
  • C: Clear all sightings when in delete mode.
  • H: Hide sighting labels.
  • B: Send bounding box back.
  • Space: Commit changes to the database, and move onto the next image.
  • Tilde(~): Go back to the previous image.
  • Delete or [: Decrease image brightness.
  • Insert or ]: Increase image brightness.
  • End or ;: Decrease image contrast.
  • Home or ': Increase image contrast.
  • PageDown or <: Decrease image saturation.
  • PageUp or >: Increase image saturation.
  • Backspace or /: Reset image brightness, contrast and saturation.

Some notes:
  • You cannot go back to a previous image until the sightings you submitted have been successfully registered. This should not take long, but can potentially take a little time when the server is busy. Please be patient if this is the case.
  • All changes to the labels here will affect the labels associated with the image and its cluster. Thus, if you only have lion sightings in an image who's cluster contains both lion and impala, the cluster will still be labelled with both species, but the image will now only be labelled as containing lion.
  • If an image contains nothing, simply delete all the sightings and by implication, it will be considered empty.
  • If the box associated with one sighting is on top of another, preventing you from clicking on it, simply click the send to back button or press the b key, and click on the offending box. This will send it to the back, allowing you to click on the boxes behind it.
