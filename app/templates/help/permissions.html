Welcome to the permissions page. Here you will be able to vier your permissions, manage the users in your organisation, manage surveys you have shared with other organisations and surveys that have been shared with you.

Your Permissions

Here you can view your permissions for the organisations that you are a member of. You can view the following permissions:

  • Default Access: The default access permission indicates the level of access that you have for the data in the organisation. The default access permissions will be applied by default to all data in your organisation.
  • Annotation Access: The annotation access permission indicates whether you have access to annotation jobs. The annotation access permissions will be applied by default to all annotation jobs in your organisation.
  • Survey Creation: The survey creation permission indicate whether you can create surveys in your organisation.
  • Survey Deletion: The survey deletion permission indicate whether you can delete surveys and annotation sets in your organisation.
  • Root User: The root user permission indicate whether you are the root user in your organisation.

You can also view your permission exceptions which alter your default permisions for a particular survey. You can view the following permissions:

  • Access: The access permission indicates the level of access that you have for a specific survey.
  • Annotation Access: The annotation access permission indicates whether you have access to annotation jobs for a specific survey.

Users

Here you can view all the users in the organisations that you are a member of as well as invite users to join a specific organisation. You can remove a user from an organisation by clicking the associated remove button. You can manage the following permissions for a user:

  • Default Access

    Here you can set the default access permissions for a user in your organisation. The default access permissions will be applied by default to all data in your organisation. The access levels are as follows:

    Access Type Jobs Only View Surveys and Results Launch Annotation Jobs Add/Edit Annotation Sets Edit Survey Add/Remove Users Share Surveys Edit Permissions
    Worker Yes No No No No No No No
    Hidden No No No No No No No No
    Read No Yes No No No No No No
    Write No Yes Yes Yes Yes No No No
    Admin No Yes Yes Yes Yes Yes Yes Yes
  • Annotation Access

    Here you can set whether a user will have access to annotation jobs. If a user has access to annotation jobs they will be able to annotate surveys.

  • Survey Creation

    Here you can set whether a user will be able to create surveys. If a user has access to survey creation they will be able to create surveys as well as specify the permission for users in the organisation for the survey (excluding themselves and admin users).

  • Survey Deletion

    Here you can set whether a user will be able to delete surveys. If a user has access to survey deletion they will be able to delete surveys as well as annotation sets.

  • Permission Exceptions

    Here you can setup survey exceptions for a user, overriding the default permissions for a user for a particular survey.

Data Sharing

Here you can view all the surveys that you have shared with other organisations. You can remove a survey that you have shared with another organisation by clicking the associated remove button. You can manage the access of a shared survey by selecting what you would like their access level to be. The access levels are as follows:

  • Hidden: The organisation has no access to the survey.
  • Read: The organisation can view the survey and its results but cannot edit or delete the survey.They cannot create, delete or edit annotation sets or launch annotation jobs.
  • Write: The organisation can view the survey and its results and can edit the survey. They cannot delete the survey. They can create or edit annotation sets and launch annotation jobs. They cannot delete annotation sets or annotation jobs.

You can also view all the surveys that have been shared with you by other organisations. You can remove a survey that has been shared with you by clicking the associated remove button. You can view the access permission that the organisation has given you for the survey.

You can share a survey with another organisation by selecting the 'Share Survey' button and following the instructions.
