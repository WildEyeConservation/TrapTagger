Naive Activity Analysis

The Naive Activity Analysis provides insights into the total counts for selected data combinations (species and sites) as they vary over time of day. This analysis is valuable for identifying the peak activity times of species. You can add multiple data combinations to compare the activity patterns of different species.

Key features of the Naive Activity Analysis include:

  • Data Unit: Customize the level of data aggregation (e.g., clusters, images, sightings, x time to independence).
  • Normalisation: Adjust for variations in data by normalising the counts.
  • Data Combination: Select the species and site combination for analysis.
  • Chart Type: Choose the most suitable chart type for data visualization.

You can export the generated chart as a PNG image by clicking the Export button. Further chart customization can be done in the Chart Options settings.
