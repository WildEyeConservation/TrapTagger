Activity Pattern

The Activity Pattern Analysis is a statistical tool used to determine the activity pattern of a species within a 24-hour period. It's based on the number of detections per hour of the day and is particularly useful for understanding when species are active. Multiple species can be selected to visualize how their activity patterns overlap.

Key features of the Activity Pattern Analysis include:

  • Multiple Species: You can analyze the activity patterns of multiple species simultaneously, with the option to estimate overlap between two species.
  • Axis Centering: Choose to center the graph axis around midday or midnight, depending on your preference.
  • Result Types: You can view results as either the frequency of species detections or the density of detections.
  • Time Reference: The analysis can be conducted using either solar time (based on average location) or clock time (based on time of day).
  • Export Graph: You can export the Activity Pattern graph as a PNG file by clicking the Export button.
  • Script and Data: Access the underlying R script by clicking View R Script, and download the data used in the analysis as a CSV file by clicking Download CSV. This allows for further analysis or customization of the script.

The Activity Pattern Analysis relies on the following R packages:

  • activity: Used to calculate the activity pattern of species by estimating their activity level per hour of the day.
  • overlap: Used to calculate overlap between two species, providing an overlap coefficient based on their activity patterns.
  • lubridate: Employed for date and time calculations, ensuring that detection date and time are in the correct format.
  • solartime: Utilized to calculate sunrise and sunset times based on the average location of detection sites.
  • ggplot2: Used for creating the Activity Pattern graph.

To generate Activity Pattern results, click the Generate Results button. Please note that the analysis may take some time to complete. Do not navigate away from the page or refresh it while the results are being generated.
