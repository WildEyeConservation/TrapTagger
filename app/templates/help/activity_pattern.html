Activity Pattern

The activity pattern analysis is a statistical analysis that is used to determin the activity pattern of a species in a 24 hour period. The analysis is based on the number of detections per hour of the day. Multiple species can be selected to give an overview of when the activity pattern of the species overlap. On the graph the average sunrise and sunset time is shown as a vertical line to give an indication of the time of day. Activity patterns can be used to view predator-prey relationships, foraging behaviour, and the effect of human disturbance on the activity of a species as well as many other things.

For the activity pattern analysis a date range can be selected to limit the analysis to a specific period. The analysis can be done for multiple species at the same time. Please note that if you select to do overlap estimation only two species can be selected. THe overlap estimation will show the overlap between the two species in the graph. The axis can be either centered around midday or midnight. The results can either showcase the frequercy of detections of the species or the density of detections. The analysis can either be done using solar time or clock time. Solar time is based on the average location of the sites where the species were detected. Clock time is based on the time of day. The activity pattern graph can be exported as a PNG file by clicking on the Export button.

The activity pattern analysis was calculated using an R script. The script can be viewed by clicking the View R Script button. The data used in the analysis can be downloaded as a CSV file by clicking the Download CSV button. A function is created in the R script where the CSV file can be loaded into R and the analysis can be done using the function if further analysis is required or modifications to the analysis is needed for your own purposes.

The activity pattern analysis is based on the following packages:

  • activity

    • The activity package is used to calculate the activity pattern of the species. It estimates the activity level of the species per hour of the day.

  • overlap

    • The overlap package is used to calculate the overlap between two species. It calculates an overlap coefficient based on the activity pattern of the two species.

  • lubridate

    • The lubridate package is used for date and time calculations. The package is used to convert the date and time of the detections to the correct format.

  • solartime

    • The solartime package is used to calculate the sunrise and sunset time of the detections. The sunrise and sunset time is calculated based on the average location of the sites where the species were detected.

  • ggplot2

    • The ggplot2 package is used to plot the activity pattern graph.

Please click the Generate Results button to generate the activity pattern results. Please note the activity pattern results may take a while to generate. Do not navigate away from the page or refreshh the page while the results are being generated.
