Welcome to the Analysis page. Here, you can apply various analyses to your data and customize their parameters to gain insights from your survey.

Datasets

The Datasets section allows you to select the datasets you want to analyze. You have the option to select 'All' surveys. If you have selected 'All' surveys, you have to specify which annotation set to use for surveys that have multiple annotation sets. You can also select specific surveys and annotation sets.

A date range filter can be specified to use across all analyses. Please note if no date range is specified, the full date range of the selected surveys will be used.

Sites

The Sites section allows you to select the sites you want to analyze. You have the option to select 'All' sites or specific sites. You can select sites using the following methods:

  • Search: Search for sites by name or use the advanced search to search by regular expression. Once you have found the sites you want to select, click 'Select All' or select individual sites.
  • Select on Map: Select sites by drawing a polygon on the map. Once you have drawn the polygon, click 'Save' to select the sites within the polygon.
  • Load from Group: Select sites from a previously saved site group. Click 'Load Groups' to view your saved site groups and select the group you want to use.

The sites that will be used in the analyses will appear in the 'Sites' section. Only the checked sites will be used. You can uncheck sites to remove them from the analyses. If you wish you can create a new site group from the selected sites by clicking 'Save Sites as Group'.

Analysis

The Analysis section allows you to choose from different analyses and configure their parameters. You can specify the species, sites, date range, data unit, and other necessary settings for each analysis. The following analyses are available:

  1. Summary Analysis: Provides a general data summary along with diversity, abundance, and camera trap statistics.
  2. Temporal Analysis: Offers insights into species activity patterns over time for specific species and site combinations.
  3. Spatial Analysis: Generates a heatmap of species counts at each site, allowing species and site selection.
  4. Numerical Analysis: Provides site or survey-based species counts.
  5. Time Analysis: Analyzes species counts as a function of time, with species and site selection.
  6. Activity Pattern Analysis: Presents density plots of species over time of day.
  7. Occupancy Analysis: Calculates occupancy and detection probabilities for species at each site.
  8. Spatial Capture-Recapture: Estimations of density, abundance, and detection probabilities based on identified individuals.

Chart

In the Chart section, you can customize how your charts are displayed. Configure background colours, font colours, axis colours, borders, legends, and gridlines for your charts. Your selected chart options will be applied when exporting the charts. Note that specific chart types may have limitations on certain options.
