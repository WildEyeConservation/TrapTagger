Occupancy Results

The Occupancy Results are presented in the form of two graphs and prediction tables. The content and number of graphs depend on the best occupancy model used:

Null Model: If the best occupancy model is the Null model, meaning no covariates are used, you'll see two graphs:

  • Occupancy Probability on a Site Level: This graph displays the probability that a species is present at each site.
  • Detection Probability on a Site Level: This graph shows the probability that a species is detected at each site, given that it is present.
  • Prediction Table: This table displays the predicted occupancy and detection probability for each site.

Model with Covariates: If the best occupancy model includes covariates in the formula, two sets of graphs will be displayed for each covariate:

  • Predicted Occupancy/Detection Probability on a Site Level for Covariates: This graph illustrates the predicted occupancy or detection probability at each site based on the covariate values for those sites.
  • Occupancy/Detection Probability as a Function of the Covariate: This graph depicts how the occupancy or detection probability changes as the covariate values vary.
  • Prediction Table: This table displays the predicted occupancy or detection probability for each site.

Occupancy Probability: This represents the probability that a species is present at a given site.

Detection Probability: This is the probability that a species is detected at a site, given that it is present.

You can export the graphs as images by selecting the Export button. You can copy the data from the Prediction Table by selecting the clipboard icon.
