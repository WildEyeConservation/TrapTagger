To create a new group or edit an existing group, follow these steps:

  1. Name: Enter a unique name for your group in the Name field.
  2. Description: Optionally, add a description for your group in the Description field.
  3. Add Sites: In the Add Sites section, you have several options to include sites in your group:
    • Search by Name: Search for sites by name.
    • Advanced Search: Use regular expressions for more advanced site searching.
    • Draw on Map: Select the Draw on map option to create a custom area around the desired sites.
  4. Select Sites: All the selected sites will appear at the bottom of the Add Sites section. The selected sites will appear in the Sites section.
  5. Save: Once you have selected your sites and named your group, press the Save button to create and save your group. Note only selected sites will be saved in the group
