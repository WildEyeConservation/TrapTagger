Spatial Capture-Recapture

Spatial Capture-Recapture provides comprehensive abundance estimates, detection probabilities, and sigma estimates for a species, leveraging spatial information to enhance accuracy.

The Spatial Capture-Recapture summary includes the following key components:

  • SCR Data Summary

    Provides a summary of data used in the SCR analysis, including the number of individuals, sampling occasions, and sites. It also includes mean maximum distance move (MMDM) and half the mean maximum distance move (HMMDM) per individual. A zero MMDM may indicate incomplete data or issues with site coordinates.

  • AICc

    Presents AICc values for each model, allowing model comparisons. The model with the lowest AICc value is considered the best fit.

  • Density per 100km2

    Offers density estimates per 100km2 for the best model, along with standard errors and 95% confidence intervals.

  • Abundance

    Displays abundance estimates for the best model, including standard errors and 95% confidence intervals.

  • Detection Probability

    Shows detection probability estimates for the best model, along with standard errors and 95% confidence intervals.

  • Space use (sigma)

    Provides space use estimates for the best model, including standard errors and 95% confidence intervals.

Customize your Spatial Capture-Recapture analysis by selecting observation window days, and selecting the species you want to evaluate. You can also incorporate site-level covariates and individual characteristics, such as sex, into your analysis.

The Spatial Capture-Recapture analysis was executed using an R script. You can view the script by clicking the View R Script button. Additionally, download the data used in the analysis as a CSV file by selecting the Download CSV button. The R script includes a function that allows you to load the CSV file into R for further analysis or modifications to suit your needs.

This analysis relies on the following R packages:

  • oSCR

    oSCR is used to create spatial capture-recapture models and calculate species abundance estimates.

  • raster

    The raster package is employed to create a raster object from site coordinates for plotting density maps.

  • sf

    sf is used to create a spatial object from site coordinates for plotting density maps.

Please note that Spatial Capture-Recapture operates on an individual level, so complete individual ID is essential for accurate results. Ensure site coordinates are provided for spatial analysis. To initiate the analysis, click the Generate Results button. Be patient, as Spatial Capture-Recapture results may take some time to generate. Avoid navigating away from or refreshing the page while results are processing.
