Spatial Capture-Recapture Summary

Spatial Capture-Recapture analysis provides comprehensive abundance estimates, detection probabilities, and sigma estimates for a species, leveraging spatial information to enhance accuracy. The Spatial Capture-Recapture summary includes the following key components:

  • SCR Data Summary

    Provides a summary of data used in the SCR analysis, including the number of individuals, sampling occasions, and sites. It also includes mean maximum distance move (MMDM) and half the mean maximum distance move (HMMDM) per individual. A zero MMDM may indicate incomplete data or issues with site coordinates.

  • AICc

    Presents AICc values for each model, allowing model comparisons. The model with the lowest AICc value is considered the best fit.

  • Density per 100km2

    Offers density estimates per 100km2 for the best model, along with standard errors and 95% confidence intervals.

  • Abundance

    Displays abundance estimates for the best model, including standard errors and 95% confidence intervals.

  • Detection Probability

    Shows detection probability estimates for the best model, along with standard errors and 95% confidence intervals.

  • Space use (sigma)

    Provides space use estimates for the best model, including standard errors and 95% confidence intervals.
