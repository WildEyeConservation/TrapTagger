Import Covariates

In this section, you can effortlessly import covariates from a CSV file to enhance your analysis.

Follow these steps to import covariates:

  1. Upload CSV File: Begin by clicking the Choose File button to select and upload your CSV file.
  2. Specify Columns: To ensure successful import, specify the following columns in your CSV file:
    • Site: Column containing site identifiers.
    • Latitude: Column with latitude values.
    • Longitude: Column with longitude values.
    • Site Covariates: Columns for site-specific covariates.
    • Detection Covariates: Columns for detection-specific covariates.
  3. Import: Finally, click the Import button to initiate the covariate import process.
