Here you can view the R script that was use to generate the results for the selected analysis. You can copy the script to your clipboard by clicking on the Copy button. The script can then be pasted into your R console to run the analysis yourself. You can download the data used in the analysis by clicking on the Download CSV button. The data will be downloaded as a CSV file. The script is provided for transparency and reproducibility purposes as well as to allow you to modify the script to suit your needs if you wish to do so.
