Occupancy analysis

Occupancy Analysis is a powerful statistical tool that allows you to gain insights from your camera trap data. This analysis helps determine the proportion of areas or sites occupied by a specific species. Key features of Occupancy Analysis include:

  • Occupancy Probability: It calculates the likelihood of a species being present at a given site. This is essential for understanding species distribution.
  • Detection Probability: It estimates the probability of detecting a species at a site, assuming it is present. This is useful for assessing the effectiveness of detection methods.
  • Covariate Effects: You can explore how various covariates impact both occupancy and detection probability, providing deeper ecological insights.

You have the flexibility to specify the species you would like to evaluate. Additionally, you can set the observation window (sampling period) in days.

To incorporate covariates in your analysis, click the Enter Covariates button, where you can input or import your covariates on a site level.

The Occupancy Analysis is executed using an R script, which you can view by clicking the View R Script button. You can also download the data used in the analysis as a CSV file by clicking the Download CSV button. This enables further analysis or modifications using R.

The analysis is dependent on the following R packages:

  • unmarked: Used for occupancy model creation and probability calculations.
  • camtrapR: Provides a site operation matrix used in occupancy analysis.
  • AICcmodavg: Used for model selection based on AICc values.
  • lubridate: Handles date and time calculations.
  • ggplot2: Used for plotting occupancy graphs.

To generate the Occupancy Analysis results, click the Generate Results button. Please be patient, as the analysis may take some time to complete, especially with large datasets. Avoid navigating away from the page or refreshing it during the process.
