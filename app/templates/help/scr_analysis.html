Spatial Capture-Recapture

Spatial Capture-Recapture analysis provides the following results based on individuals identified in your dataset:

  • Capture-Recapture

    Provides a summary of the abundance estimates for your species on capture-recapture models.

  • Spatial Capture-Recapture Summary

    Provides comprehensive abundance estimates, detection probabilities, and sigma estimates for a species, leveraging spatial information to enhance accuracy.

  • Spatial Captures

    A graph of the average spatial locations of the individuals identified in your dataset.

  • State Space

    The state space used in the analysis.

  • Density Map

    Provides a estimated density heatmap for the selected species.

  • Individual Counts

    Provides a heatmap of the number of individuals detected at each site.

Customize your Spatial Capture-Recapture analysis by selecting observation window days, and selecting the species you want to evaluate. You can also incorporate site-level covariates and individual characteristics, such as sex, into your analysis. You can choose to use the state space that is calculated by default or select to mask your own state space. You can mask your own state space by uploading the corresponding shapefiles or by drawing a polygon on the map. Please note that all applicaple sites should be included in the state space otherwise an error will occur.

The Spatial Capture-Recapture analysis was executed using an R script. You can view the script by clicking the View R Script button. Additionally, download the data used in the analysis as a CSV file by selecting the Download CSV button. The R script includes a function that allows you to load the CSV file into R for further analysis or modifications to suit your needs.

This analysis relies on the following R packages:

  • oSCR

    oSCR is used to create spatial capture-recapture models and calculate species abundance estimates.

  • Rcapture

    This package is used to create capture-recapture models and calculate abundance estimates for species.

  • raster

    The raster package is employed to create a raster object from site coordinates for plotting density maps.

  • sf

    sf is used to create a spatial object from site coordinates for plotting density maps.

Please note that Spatial Capture-Recapture operates on an individual level, so complete individual ID is essential for accurate results. Ensure site coordinates are provided for spatial analysis. To initiate the analysis, click the Generate Results button. Be patient, as Spatial Capture-Recapture results may take some time to generate. Avoid navigating away from or refreshing the page while results are processing.
